# Circuit Behavior Modeling (Python)

A physics-based Python program that models voltage, charge, and equivalent capacitance in series and parallel capacitor circuits.

This project demonstrates problem solving using physics formulas and programmatic modeling.

---

## Features

- Calculate equivalent capacitance for:
  - Series circuits
  - Parallel circuits
- Compute charge stored in capacitors
- Calculate voltage distribution across components
- Menu-driven user interaction

---

## Concepts Demonstrated

- Capacitance calculations
- Series & parallel circuit behavior
- Voltage and charge relationships
- Scientific problem solving with Python

---

## Technologies Used

- Python 3
- Command-line interface (CLI)
- Mathematical modeling
